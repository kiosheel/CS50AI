# CS50 AI - Introduction to Artificial Intelligence with Python

Ce dépôt contient mes travaux réalisés dans le cadre du cursus **CS50 AI** de Harvard.

## Projets inclus :
* **Degrees** : Implémentation d'un algorithme de recherche (Breadth-First Search) pour trouver le degré de séparation entre deux acteurs.
* **Knights (Logic)** : Résolution de problèmes de logique chevaliers/valets en utilisant le calcul propositionnel.

## Compétences :
* Algorithmes de recherche.
* Logique propositionnelle.
* Modélisation de connaissances.
